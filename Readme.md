# SimHt
### Simple Http/s server without dependecies


## Description:
A file sharing app using http, as an alternative to ftp.
Preferably to be used only in local network.

To do:

To consider:

Install:
```
npm install -g simht
```

Run with:
```
simht <path to serve>
```

Example:
```
simht ./
```
