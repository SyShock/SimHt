# SimHt

### Simple Http/s server without dependencies


## Description:

A file sharing app using http, as an alternative to ftp.
Preferably to be used only in local network.

Also:

Install:
```
npm install -g https://github.com/syshock/simht
```

Run with:
```
simht <path to serve>
```

Example:
```
simht ./
```
